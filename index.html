Quick Start Guide

Get your performance review system up and running in about 30 minutes.

Step 1: Create Google Sheet (5 minutes)

  1. Create a new Google Sheet named "Performance Review System"
  2. Run the setup() function in Apps Script to create all sheets
  3. Add your employee data to the Employees sheet

Step 2: Create Google Forms (15 minutes)

Create 3 forms following the detailed guide in FORM_SETUP.md:

  • Peer Reviewer Selection Form
  • Manager Confirmation Form
  • Performance Review Form

Step 3: Configure Apps Script (5 minutes)

  1. Copy Code.gs into Apps Script
  2. Update form IDs in CONFIG section
  3. Update FORM_ENTRY_IDS with your actual entry IDs
  4. Run setup() to initialize

Step 4: Set Up Triggers (3 minutes)

Configure automated triggers for form submissions and reminders. See TRIGGERS_SETUP.md for details.

System Architecture

The system uses a multi-sheet architecture for better organization:

Sheet Structure

  • Employees Sheet: Master employee data (7 columns)
  • Peer Selection Sheet: Phase 1 - Employee peer selection (16 columns)
  • Manager Confirmation Sheet: Phase 2 - Manager confirmation (18 columns)
  • Review Assignments Sheet: Phase 3 - All review assignments (9 columns, one row per review)
  • Dashboard Sheet: Optional summary and reporting

Form Setup

Three Google Forms are required. Important: Google Forms does not support truly hidden fields - fields will be pre-filled but visible to users.

Form 1: Peer Reviewer Selection

  • Employee Email (pre-filled)
  • Employee Name (pre-filled)
  • Select Peer Reviewers (must select exactly 5)

Form 2: Manager Confirmation

  • Employee Email (pre-filled)
  • Employee Name (pre-filled)
  • Confirm/Reselect Peer Reviewers (checkbox list)

Form 3: Performance Review

  • Reviewer Email (pre-filled)
  • Reviewee Email (pre-filled)
  • Review Type (pre-filled)
  • Your standard review questions

  • How employees select peer reviewers

  • How managers confirm or modify selections
  • How confirmed peers feed into review assignments

Complete Review Cycle

  1. Week 1: Run sendPeerSelectionInvites() - Employees select peers
  2. Week 2: Run sendManagerConfirmationEmails() - Managers confirm
  3. Week 3: Run sendReviewAssignments() - Reviews assigned
  4. Ongoing: Automated reminders sent via triggers
